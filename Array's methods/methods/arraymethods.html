Array Methods Definitions

  • push:

    Adds one or more elements to the end of an array.

    Array ke end me ek ya zyada elements add karta hai.

  • pop:

    Removes the last element from an array.

    Array se last element ko remove karta hai.

  • shift:

    Removes the first element from an array.

    Array se pehla element remove karta hai.

  • unshift:

    Adds one or more elements to the beginning of an array.

    Array ke beginning me ek ya zyada elements add karta hai.

  • concat:

    Combines two or more arrays into a single array.

    Do ya zyada arrays ko ek single array me combine karta hai.

  • slice:

    Returns a shallow copy of a portion of an array into a new array.

    Array ka ek hissa nikal kar ek naye array me copy karta hai.

  • splice:

    Changes the contents of an array by removing or replacing elements.

    Array ke elements ko remove ya replace karta hai.

  • indexOf:

    Returns the first index at which a given element can be found in the array.

    Array me kisi given element ka pehla index return karta hai.

  • includes:

    Checks if an array contains a certain element.

    Check karta hai ki array me koi specific element hai ya nahi.

  • reverse:

    Reverses the elements of an array in place.

    Array ke elements ko ulta kar deta hai.

  • join:

    Joins all elements of an array into a string.

    Array ke sabhi elements ko ek string me join karta hai.

  • split:

    Splits a string into an array of substrings.

    Ek string ko substrings ke array me split karta hai.

  • sort:

    Sorts the elements of an array in place.

    Array ke elements ko sort karta hai.

  • find:

    Returns the first element in the array that satisfies a provided testing function.

    Array me pehla element return karta hai jo testing function ko satisfy kare.

  • findIndex:

    Returns the index of the first element that satisfies a provided testing function.

    Pehle element ka index return karta hai jo testing function ko satisfy kare.

  • map:

    Creates a new array with the results of calling a provided function on every element.

    Har element par ek function call karke ek naye array ko create karta hai.

  • reduce:

    Executes a reducer function on each element of the array, resulting in a single output value.

    Array ke har element par ek reducer function execute karke ek single output value produce karta hai.

  • some:

    Tests whether at least one element in the array passes the test implemented by the provided function.

    Check karta hai ki kya array me kam se kam ek element provided function ke test ko pass karta hai.

  • every:

    Tests whether all elements in the array pass the test implemented by the provided function.

    Check karta hai ki kya array ke sabhi elements provided function ke test ko pass karte hain.

  • at:

    Returns the element at a given index, allowing negative indices to count from the end.

    Diye gaye index par element return karta hai, negative indices ko end se count karne ki suvidha deta hai.
