Question 1: Show/Hide Password

Ek input field banao jisme password type ka data enter ho. Uske paas ek button banao jise click karne par password show ya hide ho jaye. Jab password show ho raha ho tab button ka text "Hide Password" ho aur jab password hide ho tab button ka text "Show Password" ho.

Question 2: Change Color on Button Click

Ek button aur ek div banao. Jab button ko click kiya jaye, toh div ka background color randomly change ho jana chahiye.

Question 3: Simple Image Slider

Do buttons aur ek image element banao. Pehla button "Previous" aur dusra button "Next" ho. Jab "Next" button par click kiya jaye, toh image next image mein change ho. Jab "Previous" button par click kiya jaye, toh image previous image mein change ho. Tumhe kuch images ke URLs store karne honge.

Question 4: Count Down Timer

Ek button aur ek paragraph banao. Jab button par click kiya jaye, toh paragraph mein 10 se count down shuru ho. Har second ke baad count 1 se kam hona chahiye. Jab count 0 ho jaye, toh paragraph mein "Time's Up!" likha dikhana chahiye.
